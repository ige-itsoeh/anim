¿Qué es Reveal.js?

Es un framework de código abierto para crear presentaciones interactivas usando HTML, CSS y JavaScript.

  • Fácil de usar y personalizar.
  • Funciona en cualquier navegador.
  • Permite transiciones y efectos dinámicos.
